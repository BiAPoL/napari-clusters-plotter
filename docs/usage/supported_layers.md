# Supported layers

The napari-clusters-plotter is designed to work with features originating from a range of napari layer types. Currently, the napari-clusters-plotter supports the following layers:

- [Labels layer](https://napari.org/stable/howtos/layers/labels.html)
- [Points layer](https://napari.org/stable/howtos/layers/points.html)
- [Vectors layer](https://napari.org/stable/howtos/layers/vectors.html)
- [Surface layer](https://napari.org/stable/howtos/layers/surface.html)
- [Shapes layer](https://napari.org/stable/howtos/layers/shapes.html)
